# The AVC Files Codex

This folder collects the narrative building blocks for **Decrypt The Girl**, organized into discreet layers so the myth, the manuscript, and the systems that bind them remain easy to navigate.

## Directory Map
- `README.md` — This guide to the layout.
- `LICENSE.md` — Licensing for the archival materials (add when ready).
- `.gitignore` — Workspace hygiene for drafts and exports (add when needed).
- `codex/` — Atomic nodes, constellation maps, and rooms used to assemble the story.
  - `nodes/` — Each poem or section lives in its own markdown file with frontmatter.
  - `chapters/` — JSON files that order nodes into linear chapters.
  - `rooms/` — Interactive interfaces mapped to narrative beats.
  - `constellations/` — Thematic or nonlinear maps that re-link nodes.
  - `manifest/` — Indexes and shared constants for tags, timelines, and modes.
- `manuscript/` — Reader-facing exports for the linear book experience.
- `systems/` — Reference frameworks and schemas underpinning the interactive layers.
- `assets/` — PDFs, images, and decks that accompany the codex.
- `legal_archive/` — Redacted legal artifacts stored for provenance.
- `ops/` — Business and monetization scaffolding.

Use this structure when adding new poems or system artifacts so every piece of the codex can be referenced across maps, rooms, and manuscript exports.
